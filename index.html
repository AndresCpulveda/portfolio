Skills

  • HTML

    • HTML is very simple and fundamental in the web development but is very important to undetstand it deeply, since it has many tags and corresponding attributes, to know them well garanties a fine, structured and semantic code also very important for web SEO

  • CSS

    • CSS is full of posibilities, you will always discover new things in this language, understand CSS deeply allows for faster development when creating an interface from scratch or when changes must be made. A devolper proficient in CSS can easily meet the expectations and requirements to create a unique design, different from everything else. Personally I find it extremely usefull to use the Tailwind CSS library to apply styles in a more efficient way, many times combined with pure CSS for most specific needs.

  • JavaScript

    • Without a doubt, JavaScript is the fundamental language of frontend development and for this very reason, it is the language that I focused on the most from the beginning of my learning, I currently master it with ease in its most recent versions of ECMAScript, as well as many of its internal APIs such as DOM, Web Storage, IndexedDB, etc...

  • Git & GitHub

    • Version control is essential for any project of any magnitude, that's why I use git and github for all my developments, the practicality and utility that all git functions provide allows me to work much more fluidly and productively.

  • React JS

    • I decided to learn React for its practicity and great adoption, from the first moment I realized the useful it is compared to programming in Vanilla JavaScript thanks to its use of components and hooks, I have deeply enjoyed the workflow and scalibality of my projects (like my Planner App and my CRMs) when implementing this framework. React has many things to offer.
