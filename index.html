Lenguajes & tecnologias

  • HTML

    • Si bien HTML puede ser un lenguaje de marcado facil y rapido de aprender, este cuenta también con muchas etiquetas y características poco usadas como el uso de HTML semántico que puede llegar a ser muy util, en especial en el desarrollo web.

  • CSS

    • CSS es un lenguaje lleno de posibilidades, donde siempre estoy descubriendo cosas nuevas, es quizas el lenguaje al que más tiempo le empleé durante mis inicios aprendiendo desarrollo frontend, antes de especializarme utilizando frameworks como Tailwind, esta pagina de portafolio esta estilada en su totalidad con Tailwind. Es un lenguaje con el que tengo basta seguridad y comodidad.

  • JavaScript

    • Sin duda alguna, JavaScript es el lenguaje fundamental del desarrollo frontend y por esto mismo, es el lenguaje en que más me he enfocado desde el inicio de mi aprendizaje, actualmente lo domino con facilidad en sus versiones mas recientes de ECMAScript, al igual que muchas de sus APIs internas como DOM, Web Storage, IndexedDB, Notifications Api, etc...

  • Git & GitHub

    • El control de versiones es algo fundamental para todo proyecto de cualquier magnitud, por eso empleo git y github para todos mis desarrollos, la practicidad y utilidad que brindan todas las funciones de git me permite un trabajo mucho mas fluido y productivo.

  • Fluido Ingles C1

    • Adicionalmente a las tecnologías, tengo un nivel de ingles fluido que me permite comunicarme con facilidad en este idioma y más importante, me permite consumir todo tipo de contenido. Esto me facilita entender documentación que no este traducida y tener muchas opciones a la hora de adquirir nuevo conocimiento.

Aprendizaje

Desde que inicié mi aprendizaje quería dominar las tecnologías más actualizadas y utilizadas actualmente.
Me di cuenta que la industria avanza rápidamente y que el único lugar donde siempre podría encontrar nuevo conocimiento y mantenerme a la vanguardia de las nuevas herramientas era en internet.
Por esto mismo decidí apropiarme de mi aprendizaje desde un enfoque autodidacta, encontrando los recursos de mayor calidad y más completos que hay en el internet. Durante todo este tiempo he completado multiples certificaciones y sobre todo una gran cantidad de horas empleadas poniendo en práctica y profundizando cada concepto nuevo que aprendo. Desde lo más fundamental y básico, hasta lo más práctico y funcional.
